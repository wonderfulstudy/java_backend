### 封装方法

获取或设置成员属性使用setXxx()、getXxx()方法；

类中的所有属性必须使用private修饰，获取方法必须使用内部方法。

### 构造方法

-   构造方法名称必须与类名称保持一致
-   构造方法不允许设置任何的返回值类型
-   构造方法是使用关键字new实例化对象的时候自动调用的
